# proftpd_for_backup
Проект бекап
Сервер Proftpd

Нужно указать свой пароль в
FTPUSER_PASSWORD_SECRET=password

Юзер указан в 
FTPUSER_NAME=ftpuser

Сборка проекта
./start_proftpd.sh

Удаление проекта
./delete_proftpd.sh

Файл конфигурации proftpd
proftpd.conf.j2

Файл конфигурации tls/ssl
tls.conf.j2
